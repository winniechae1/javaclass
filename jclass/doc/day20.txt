# day20 

Map 
하나의 데이터를 그 데이터를 구분할 수 있는 키값과 한쌍으로 만들어서 데이터를 기억하는 컬렉션의 한 종류이다.
그리고 사용할 경우에는 키값을 이용해서 데이터를 꺼낼 수 있도록 한 것이다.

Map 계열은 많은 양의 데이터를 보관하는 용도보다는 데이터를 손쉽게 구분하여 사용할 목적으로 사용된다.

HashTable(구 버전)/ HashMap(최근 버전) 
	
키와 값을 하나의 쌍으로 묶어서 저장하는 컬렉션 클래스
키는 중복될 수 없지만 값은 중복을 허용한다.

주요 함수 
Object put(Object key, Object value) : Map에 value 객체를 key 객체에 연결하여 저장한다.
Object get(Object key) : 지정한 key객체에 대응하는 value 값을 찾아서 반환한다.

values() :  Map에 저장된 모든 value객체를 반환한다.
keys() : Map에 저장된 모든 key객체를 반환한다. (HashTable에만 존재하는 함수)
Set keyset() : Map에 저장된 모든 key객체를 반환한다. (HashTable, HashMap 둘다 사용할 수 있다. Map계열에서는 모두 사용가능)
entrySet() : Map에 저장된 key - value 쌍을 Map.Entry 타입의 객체로 저장한 Set으로 반환한다.

주의]
	Map은 항상 key-value 두개가 이써야 저장된다. Set은 value 한개만 있으면 저장이 된다.
	따라서 key-value를 하나로 묶어야만 Set에 저장된다.
	
	이때 이 문제를 해결하기 위한 클래스(VO 클래스)가 존재한다. 그 클래스가 바로 Map.Entry 
==>	이 클래스 안에는 key와 value 변수를 은닉화 시켜놓고 이 변수를 사용할 수 있는 함수를 
	제공함으로써 두개의 변수를 하나로 묶어 놓은 클래스이다.
	
TreeMap
TreeSet도 마찬가지로 내부에서 정렬하면서 데이터를 기억하는 Map이다.
이때 정렬 기준은 키값을 이용해서 정렬하게 된다. 따라서 키값이 정렬되지 않는 클래스는 사용할 수 없고 
필요하다면 정렬 방식을 정해서 사용해야 한다.

Tmp abc() {
	int a = 10;
	String b = "홍길동";
	Tmp t = new Tmp();
	t.a = a;
	t.b = b;
	
	return t;
}
여러개의 데이터를 하나로 묶기 위한 목적으로 만들어 놓은 클래스 

class Tmp {
	int a;
	String b;
}
==> 빈 클래스 Value Object(VO)라고 하고 DTO(Data Transfer Object)라고도 부른다.

참고]
	컬렉션 안에는 다시 컬렉션을 입력할 수 있다.컬렉션 안에는 모든 클래스를 입력할 수 있으므로 저장이 가능하다.

제너릭스
컬렉션은 모든 형태의 데이터를 다 입력받을 수 있다.
하지만 실무에서는 한가지 형태로 입력받는 것을 원칙으로 한다.
이때 꺼낼때는 항상 형변환을 해서 꺼내야 한다.
즉, 한가지 형태로 데이터를 입력해 놓았는데 꺼낼때는 자동 형변환이 일어나지 않기 때문에 항상 강제 형변환을 해줘야 한다.

제너릭스란 
컬렉션에 입력하는 데이터를 미리 알려줌으로써 나중에 꺼내서 쓸 때 좀 더 편하게(강제형변환 없이) 꺼낼 수 있도록 해주는 것

형식]
	ArrayList<String>	list = new ArrayList<String>();
	list에는 문자열 데이터만 입력할 수 있게 된다. 꺼낼 때는 자동으로 문자열 타입으로 꺼내게 된다.

참고]
	반드시 컬렉션마다 모두 제너릭스 선언을 해줘야 한다.

참고]
	제너릭스는 전파되지 않는다. 하나의 컬렉션이 제너릭스 선언이 되었더라도 그 컬렉션을 이용하는 다른 컬렉션은 
	제너릭스를 잊어버린다. 따라서 새로운 컬렉션도 다시 제너릭스 선언을 해줘야 한다.

참고]
	특수한 제너릭스 - 원래 제너릭스는 특정 클래스만 입력받을 목적으로 사용된다.
	
	형식]
		<? super 클래스이름>
		==> 지정한 클래스 포함 상위 클래스까지 허락
		
		<? extends 클래스이름>
		==> 지정한 클래스 포함 하위 클래스까지 허락
		
	이 방법은 제너릭스를 선언하지 않은 것과 동일한 효과이다.
